# crypto

libcrypto bindings for Chicken v5. Inspired by bindings in other popular
interpreted languages e.g Python, Node.js.

## roadmap

### v1

* [ ] Public / private / secret key interface
* [ ] MACs
* [ ] Digests
* [ ] Symmetric Algorithms ("ciphers")
* [ ] Asymmetric Algorithms
* [x] Signatures
* [ ] Key Derivation

### v2
* [ ] Key wrapping
